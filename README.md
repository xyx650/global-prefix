# global-prefix
> Get the npm global path prefix.

**This repo is forked from [jonschlinkert/global-prefix](https://github.com/jonschlinkert/global-prefix)**

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install github:xyx650/global-prefix
```

This is partially based on the code used by npm internally to resolve the global prefix.

## Usage

```js
var prefix = require('global-prefix');
//=> '/usr/local' (this path will differ by system and user-defined config)
```
